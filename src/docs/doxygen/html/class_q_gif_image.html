◆ setDefaultTransparentColor()

void QGifImage::setDefaultTransparentColor ( const QColor &  color)

Set the default transparent color.

Unlike other image formats that support alpha (e.g. png), GIF does not support semi-transparent pixels. The way to achieve transparency is to set a color that will be transparent when rendering the GIF. So, if you set the transparent color to black, the black pixels in the gif file will be transparent.
